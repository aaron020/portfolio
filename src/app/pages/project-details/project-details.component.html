Project Overview

This project emerged from my AWS certification studies, where I found converting learning material into multiple choice questions was the most effective way to prepare for exams. After filling four notepads with handwritten questions, I wanted to digitize this process. The website allows users to create their own multiple choice questions, take tests with these questions, and share their quizzes with others.

Core Functionality

  • User creation and login system.
  • User view and create topics
  • User add / update / delete questions within topic
  • User take test
